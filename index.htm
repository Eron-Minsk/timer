Invigilation Timer - Help Guide

Getting Started

  1. Enter the venue name and click "Set Venue"
  2. Add timers by entering module code and duration
  3. Use preset durations for quick setup
  4. Click "START ALL TIMERS" to begin

Timer Controls

  • Pause/Resume: Toggle individual timers
  • Reset: Return timer to original duration
  • +1m/+5m: Add extra time
  • ×: Remove timer

Global Controls

  • PAUSE ALL: Pause/Resume all timers
  • RESET ALL: Reset all timers
  • +1m/+5m ALL: Add time to all timers
  • Toggle Fullscreen: Enter/Exit fullscreen mode

Features

  • Save and load timer configurations
  • Visual and audio notifications
  • Progress bars with warning indicators
  • Maximum 10 concurrent timers
